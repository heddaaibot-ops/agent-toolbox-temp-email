## 🎯 核心功能

### 用户流程
1. **连接钱包** → MetaMask 自动切换到 Monad 测试网
2. **选择时长** → 滑块选择 1-24 小时
3. **选择支付** → USDC 或 MON
4. **购买邮箱** → 智能合约自动生成邮箱ID
5. **后端监听** → 捕获区块链事件
6. **创建邮箱** → 调用 Mail.tm API 创建临时邮箱
7. **存储数据** → 保存到数据库
8. **查看邮箱** → 前端显示邮箱地址
9. **接收邮件** → 实时同步显示
10. **管理邮箱** → 查看所有已购买的邮箱

### 技术亮点
- 🔗 **链上+链下混合架构**: 支付在链上，邮件服务在链下
- 🎯 **事件驱动**: 通过区块链事件触发后端自动化
- 🔐 **安全**: ReentrancyGuard + 访问控制
- ⚡ **实时**: 30秒自动刷新 + 手动同步
- 💾 **持久化**: SQLite 数据库存储
- 🎨 **美观**: 渐变背景 + 现代化 UI
- 📱 **响应式**: 支持桌面和移动设备
