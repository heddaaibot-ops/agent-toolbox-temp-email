# Agent's Toolbox - Monad Hackathon Project

## 🚀 核心愿景 (Core Vision)

建立一个基于 Monad 区块链的去中心化工具市场，专为 AI Agent 提供“链上资源”。Agent 可以通过微交易，安全、可信地购买和使用现实世界中的稀缺资源。

## 🎯 MVP: 一次性邮箱服务 (Disposable Email Service)

**目标:** 让 Agent 可以通过智能合约调用，付费获取一个临时的、可接收邮件的邮箱地址，用于完成各种服务的注册和验证。
